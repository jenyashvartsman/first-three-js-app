# First Three.js App

This is a simple Three.js app built with Vite and deployed to GitHub Pages.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```
3. Build for production:
   ```sh
   npm run build
   ```
4. Deploy to GitHub Pages:
   ```sh
   npm run deploy
   ```
